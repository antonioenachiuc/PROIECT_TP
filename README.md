# Pac-Man (SDL)

Deocamdata proiectul include mișcarea personajului, coliziuni, sistem de scor și o interfață grafică.

## Funcționalități

* Interfață grafică realizată cu SDL
* Mișcarea lui Pac-Man
* Detecția coliziunilor cu pereții
* Sistem de scor (crește când sunt colectate puncte)
* Miscarea Fantomelor + Coliziunea cu Pac-Man
* Power-up Cirese 

## Build

Proiectul folosește un Makefile pentru compilare și rulare.

Comenzi:

* `make` – compilează proiectul
* `make run` – rulează jocul
* `make clean` – curăță fișierele build

## Tehnologii

* C
* SDL2
